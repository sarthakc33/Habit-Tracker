# Reality Check: Productivity vs Time Tracker

A fully functional, responsive, and modern full-stack web application for task management and time tracking.

## Features

- **Task Planning System**: Daily, weekly, or specific weekday recurring tasks.
- **Time Tracking**: Real-time timer with a focus mode.
- **Analytics Dashboard**: Advanced chart visualization to compare planned vs. actual time.
- **Gamification**: XP, levels, and streaks for motivation.
- **Multi-user Authentication**: Secure JWT-based authentication.
- **Real-time Notifications**: Socket.io-powered alerts.

## Technology Stack

- **Frontend**: HTML5, Vanilla CSS, Vanilla JavaScript, Chart.js
- **Backend**: Node.js, Express.js
- **Database**: Local JSON storage
- **Authentication**: JWT & bcryptjs
- **Real-time Engine**: Socket.io

## Installation and Setup

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   cd "Habit Tracker"
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the application:**
   ```bash
   npm run dev
   ```
   The server will start on `http://localhost:3000`.
